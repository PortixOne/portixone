# @portixone/shared

Constants and error classes shared between the Portix Runtime and the SDKs. No business logic and no protocol types — those live in [`@portixone/protocol`](https://www.npmjs.com/package/@portixone/protocol).

This is an internal dependency of [`@portixone/sdk`](https://www.npmjs.com/package/@portixone/sdk) and the runtime — most developers won't import it directly.
